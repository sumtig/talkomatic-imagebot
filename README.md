# talkomatic-imagebot
can load images and export them. video support, experimental.
# how to use
(need node.js for it to actually work.) you run the bot.js file using `node bot.js`. enter a room code, and its there. you need a token from https://github.com/Talkomatic-CLI-Edition.
# how to get token
1. install talkomati cli from https://github.com/Talkomatic-CLI-Edition.
2. run the script and do what it says, it should give you a token file without extension.
3. put the token into the token field in the bot.js file.
4. have fun!
# info

example:

frame 1

frame 2

etc.
add a 'name-config.txt' when adding the animation. add a 'delay: desired number' inside it.
